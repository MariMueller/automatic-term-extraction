# Automatic Term Extraction & Human Validation

**Master's Research (2022)**

A hybrid methodology combining **Corpus Linguistics, Terminology, Computational Linguistics, and expert human validation** to improve Automatic Term Extraction (ATE) from specialized bilingual corpora.

---

## Overview

This repository presents the methodology developed during my Master's research in Language Studies, with a focus on Terminology, Corpus Linguistics, and Computational Linguistics.

The project proposes a hybrid workflow that combines Automatic Term Extraction (ATE), corpus analysis, and expert linguistic validation to identify and validate domain-specific terminology from bilingual corpora.

The methodology also incorporates Lexical Semantics to establish semantic relationships between concepts and organize terminology into hierarchical domain structures.

## Human Validation Criteria

The automatically extracted term candidates were manually validated using Lexical Semantics criteria proposed by L'Homme (2020), including:

- Domain relevance
- Argument structure
- Morphological relationships
- Semantic relationships
- Hypernymy
- Hyponymy
- Meronymy
- Synonymy

---

## Main Contributions

- Developed a hybrid Automatic Term Extraction methodology.
- Compared Automatic Term Extraction tools using specialized corpora.
- Created a gold-standard terminology dataset for evaluation.
- Applied human linguistic validation to automatically extracted term candidates.
- Organized specialized concepts into hierarchical domain trees.
- Developed a bilingual Portuguese–English terminology database containing more than **270 validated specialized terms**.
- Demonstrated how Corpus Linguistics and Terminology can improve terminology extraction accuracy.
