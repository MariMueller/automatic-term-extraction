## 🇧🇷 Português

# Glossário Léxico Bilíngue (Militar)

Este projeto consiste em uma interface web moderna e intuitiva para um glossário bilíngue focado em terminologia militar (Português-Brasil / Inglês). Os dados do glossário foram processados e estruturados para serem consumidos dinamicamente na página.

**🌐 Acesse o projeto online:** [https://glossario-lexico-bilingue.marimuller.com/](https://glossario-lexico-bilingue.marimuller.com/)

### Funcionalidades

- **Pesquisa em Tempo Real**: Campo de busca que filtra os termos do glossário instantaneamente conforme o usuário digita, buscando tanto em português quanto em inglês.
- **Visualização Dinâmica**: Os dados (termos principais e suas variações) são carregados a partir de arquivos locais de dados (`data.js` / JSON) e renderizados na tela via JavaScript de forma ágil e otimizada.
- **Design Responsivo e Moderno**: Interface limpa e agradável desenvolvida com CSS, focada na legibilidade, usabilidade e na experiência do usuário final.

### Estrutura de Arquivos

- `index.html`: A estrutura principal e os elementos da página web.
- `styles.css`: As regras de estilização, design e layout da aplicação.
- `script.js`: O motor da aplicação, responsável pela lógica de busca, filtragem e a renderização dinâmica dos termos no HTML.
- `data.js`: Arquivos contendo os dados (termos, traduções e variações) extraídos do léxico original (planilha), estruturados para o consumo fácil do front-end.

### Como Executar

Por ser uma aplicação baseada inteiramente em HTML, CSS e JavaScript estático (Vanilla JS), não é necessário nenhum servidor complexo ou instalação de dependências para visualizá-la.

1. Navegue até a pasta `glossario-lexico-bilingue`.
2. Dê um duplo clique no arquivo `index.html` para abri-lo diretamente em qualquer navegador moderno (Chrome, Edge, Firefox, Safari).
3. Utilize a barra de pesquisa para explorar e localizar as terminologias militares!

## 🇺🇸 English

# Bilingual Lexical Glossary (Military)

This project consists of a modern and intuitive web interface for a bilingual glossary focused on military terminology (Brazilian Portuguese / English). The glossary data has been processed and structured to be dynamically consumed on the page.

**🌐 Access the project online:** [https://glossario-lexico-bilingue.marimuller.com/](https://glossario-lexico-bilingue.marimuller.com/)

### Features

- **Real-Time Search**: Search field that instantly filters glossary terms as the user types, searching in both Portuguese and English.
- **Dynamic Visualization**: Data (main terms and their variations) are loaded from local data files (`data.js` / JSON) and rendered on the screen via JavaScript in an agile and optimized way.
- **Responsive and Modern Design**: Clean and pleasant interface developed with CSS, focused on readability, usability, and the end-user experience.

### File Structure

- `index.html`: The main structure and elements of the web page.
- `styles.css`: The styling rules, design, and layout of the application.
- `script.js`: The engine of the application, responsible for search logic, filtering, and dynamic rendering of terms in HTML.
- `data.js`: Files containing the data (terms, translations, and variations) extracted from the original lexicon (spreadsheet), structured for easy front-end consumption.

### How to Run

Since it is an application based entirely on HTML, CSS, and static JavaScript (Vanilla JS), no complex server or dependency installation is required to view it.

1. Navigate to the `glossario-lexico-bilingue` folder.
2. Double-click the `index.html` file to open it directly in any modern browser (Chrome, Edge, Firefox, Safari).
3. Use the search bar to explore and locate military terminologies!
